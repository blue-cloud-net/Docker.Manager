# 功能清单

> 标注说明：🟢 阶段一 · 🔵 阶段二 · 🟡 阶段三

---

## 容器管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| 容器列表 | 展示所有容器（运行中 / 已停止），含状态、镜像、端口、创建时间 | 🟢 |
| 启动容器 | 启动已停止的容器 | 🟢 |
| 停止容器 | 优雅停止正在运行的容器 | 🟢 |
| 重启容器 | 重启容器 | 🟢 |
| 删除容器 | 删除已停止的容器，选项：强制删除 | 🟢 |
| 日志查看 | 查看容器标准输出日志，支持行数限制与时间过滤 | 🟢 |
| 日志实时流 | 通过 SignalR 实时推送日志输出 | 🟡 |
| 容器详情 | 查看容器完整配置信息（Inspect） | 🟢 |
| 容器统计 | 实时 CPU、内存、网络 I/O 监控，SignalR 推送 | 🟡 |
| 交互式终端 | 通过 WebSocket 进入容器执行命令（xterm.js） | 🟡 |
| 容器异常通知 | 容器非正常退出时推送页面内通知 | 🟡 |

---

## 镜像管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| 镜像列表 | 展示本地所有镜像，含名称、标签、大小、创建时间 | 🟢 |
| 拉取镜像 | 从 Docker Hub 拉取镜像，支持指定 Tag | 🟢 |
| 删除镜像 | 删除本地镜像，选项：强制删除 | 🟢 |
| 镜像详情 | 查看镜像完整元数据（Inspect） | 🟢 |
| 镜像构建 | 上传 Dockerfile 及上下文文件进行构建 | 🟡 |
| 私有 Registry | 配置私有 Registry 地址与认证信息，支持 pull/push | 🟡 |

---

## Volume 管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| Volume 列表 | 展示所有本地 Volume，含名称、驱动、挂载点 | 🟡 |
| 创建 Volume | 创建命名 Volume | 🟡 |
| 删除 Volume | 删除未使用的 Volume | 🟡 |
| Volume 详情 | 查看 Volume 完整信息 | 🟡 |

---

## Network 管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| Network 列表 | 展示所有网络，含名称、驱动、子网 | 🟡 |
| 创建 Network | 创建自定义网络，支持指定驱动与子网 | 🟡 |
| 删除 Network | 删除未使用的网络 | 🟡 |
| Network 详情 | 查看网络完整信息及已接入的容器 | 🟡 |

---

## Docker Compose 管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| Stack 列表 | 展示通过 Compose 部署的 Stack | 🟡 |
| 部署 Stack | 上传 `docker-compose.yml` 文件部署 Stack | 🟡 |
| 更新 Stack | 重新上传 YAML 文件更新 Stack | 🟡 |
| 停止 / 删除 Stack | 停止或彻底删除一个 Stack | 🟡 |

---

## 用户与权限管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| 登录 / 登出 | 用户名密码登录，返回 JWT Token | 🔵 |
| 用户列表 | 管理员查看所有用户 | 🔵 |
| 创建用户 | 管理员创建新用户并分配资源权限 | 🔵 |
| 修改密码 | 用户修改自身密码 | 🔵 |
| 资源授权 | 按主机 / 容器粒度为用户分配访问权限 | 🔵 |
| 删除用户 | 管理员删除用户 | 🔵 |

---

## 多主机管理

| 功能 | 说明 | 阶段 |
|------|------|------|
| 本地 Socket 连接 | 通过 `/var/run/docker.sock` 管理本机 Docker | 🟢 |
| TCP 远程连接 | 配置远程 Docker API 地址（可选 TLS） | 🟡 |
| Agent 连接 | 目标主机部署 Agent，通过 Agent 管理 | 🟡 |
| 主机列表 | 展示所有已配置的 Docker 主机及连接状态 | 🟡 |
